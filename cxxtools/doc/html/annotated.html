C++Utilities Übersicht

Hier folgt die Aufzählung aller Klassen, Strukturen und Varianten mit einer Kurzbeschreibung:
  • arg< T > (Auswertung von Kommandozeilenoptionen)
  • arg< bool > (Spezialisierung von arg<T> für bool)
  • arg< const char *> (Spezialisierung von arg<T> für const char*)
  • tcp::eof (Die Klasse wird geworfen, wenn keine Zeichen gelesen werden konnten)
  • tcp::Exception (Tcp::Exception wird bei Fehlern ausgelöst)
  • tcp::iostream (Stream-Socket erweitert um Pufferung über std::iostream-Bibliothek)
  • tcp::Server (Serversocket)
  • tcp::Socket (Basisklasse für Sockets)
  • tcp::Stream (Stream repräsentiert ein Client-Socket)
  • tcp::streambuf (Streambuf ist ein std::streambuf für Socket-Verbindungen)
