Öffentliche Datenelemente

 Socket (int domain, int type, int protocol) throw (Exception)
 Konstruktor erzeugt eine Instanz und erzeugt einen Socket auf Betriebssystemebene. Bei Fehlern wird eine Exception ausgelöst.

 Socket (int fd=-1)
 Konstruktor initialisiert die Klasse mit einem bestehenden Socket.

virtual ~Socket ()
 Im Destruktor wird der Socket geschlossen, wenn er offen ist. Bei einem Fehler wird eine Fehlermeldung auf stderr geschrieben.

bool good () const
 Liefert true zurück, wenn der Socket geöffnet ist.

bool bad () const
 Liefert true zurück, wenn der Socket nicht geöffnet ist.

 operator bool () const
 Liefert true zurück, wenn der Socket geöffnet ist.

void create (int domain, int type, int protocol) throw (Exception)
 Erzeugt einen Socket auf Betriebssystemebene. Ist bereits einer geöffnet, wird dieser vorher geschlossen. Bei Fehlern wird eine Exception ausgelöst.

void close ()
 Schießt den Socket, wenn er geöffnet ist.

int getFd () const
 Liefert den Sockethandle zurück.
